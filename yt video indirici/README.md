# YouTube Video İndirici

YouTube videolarını farklı çözünürlüklerde indirmenizi sağlayan Python komut satırı aracı.

## Kurulum

1. **Gerekli kütüphaneyi yükleyin:**
   ```bash
   pip install -r requirements.txt
   ```
   
   veya direkt:
   ```bash
   pip install yt-dlp
   ```

## Kullanım

Programı çalıştırmak için terminalde şu komutu kullanın:

```bash
python youtube_downloader.py
```

## Adımlar

1. Program başladığında YouTube video linkini girin
2. Video bilgileri gösterilecek
3. Mevcut çözünürlükler listelenecek
4. İstediğiniz formatı seçin
5. İndirme klasörünü seçin
6. Video indirilecek!

## Desteklenen Link Formatları

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://youtube.com/watch?v=VIDEO_ID`
- Sadece `VIDEO_ID` (11 karakter)
